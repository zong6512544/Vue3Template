# 开发前请阅读我


## 文件夹目录：

- front : 前端 (Vue3 + Ts + Axios)
- server: 服务端

## 代码提交规范：

- feat:  新增（xxx）功能模块
- fix:   修复（xxx）功能模块
- style: 样式的调整
- test:  测试代码


## 前端相关

### 启动项目：

```cmd
    npm run open ip地址
    
    // example:
    npm run open 192.168.10.1:8080
```


## 服务端相关
### 启动项目

```cmd
    npm run dev
```
